Tutorials
=========

Interactive `Marimo <https://marimo.io>`_ notebooks demonstrating how to use
Rheedium for RHEED pattern simulation.

.. note::

   These notebooks live in the ``tutorials/`` directory as Marimo ``.py``
   files. To run them interactively:

   1. Clone the repository
   2. Install Marimo: ``pip install marimo``
   3. Navigate to the ``tutorials/`` directory
   4. Launch a notebook: ``marimo edit 01_kinematic_MgO.py``

Available tutorials
-------------------

- ``01_kinematic_MgO.py`` — Kinematic RHEED simulation for MgO
- ``02_kinematic_SrTiO3.py`` — Kinematic RHEED simulation for SrTiO\ :sub:`3`
- ``03_finite_domain.py`` — Finite-domain effects on RHEED patterns
- ``04_bi2se3_temperature.py`` — Temperature dependence for Bi\ :sub:`2`\ Se\ :sub:`3`

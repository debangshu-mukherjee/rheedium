Test Reference
==============

This page documents the test suite so that ``:see:`` cross-references in the
API docstrings (e.g. ``:func:`tests.test_rheedium...``` and
``:class:`tests.test_rheedium...```) resolve to clickable links.

It is intentionally kept out of the main navigation; it exists only to register
the test classes and functions in Sphinx's object inventory.

Interactive Ewald Sphere
========================

This page points to a lightweight Jupyter notebook rendered by ``myst-nb``. It
intentionally avoids ``rheedium`` and JAX while still matching the same
relativistic wavelength and Ewald-sphere geometry conventions used by the
package.

Edit the constants in the first notebook cell to vary the beam energy, grazing
angle, lattice spacing, and number of reciprocal rods, then rerun the notebook
in VS Code or Jupyter.
